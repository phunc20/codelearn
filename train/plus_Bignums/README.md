Lưu ý:

Python, Go và Java sẽ bị cấm trong bài luyện tập này.
Ví dụ:

Với arr = ["5091002802391387", "10279147192864129192650", "019279810924010927509175", "1826091625162576108257"] thì summary(arr) = "31385054833040435201469"
